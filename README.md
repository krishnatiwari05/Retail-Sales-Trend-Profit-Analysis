📊 Retail Sales Trend Profit Analysis

This project showcases an interactive **Retail Sales Trend Profit Analysis** built using Power BI to analyze and visualize sales performance across multiple dimensions.

🔍 Key Insights:

* **Sales by Segment:** Consumer segment contributes the highest share (~44%), followed by Corporate and Home Office.
* **Sales by Payment Mode:** COD is the most preferred payment method, followed by Online and Card payments.
* **Regional Analysis:** Sales distribution across Central, East, South, and West regions with dynamic filtering.
* **Monthly Trends:** Year-wise comparison of **Sales** and **Profit** highlighting growth patterns and seasonal fluctuations.
* **Geographical View:** State-wise sales and profit visualization using map integration.
* **Category Performance:** Office Supplies leads in total sales, followed by Technology and Furniture.
* **Sub-Category Analysis:** Top-performing products include Phones, Chairs, and Binders.
* **Shipping Insights:** Standard Class shipping dominates over other shipping modes.

⚙️ Features:

* Interactive filters (Region selection)
* Dynamic charts and visuals
* Clean and modern dashboard design
* Drill-down capability for detailed analysis

🛠 Tools Used:

* Power BI
* Excel
* MySQL
* Data Visualization Techniques
* Retail store Sales Dataset

🎯 Purpose:

This dashboard helps in understanding business performance, identifying trends, and supporting data-driven decision-making.
